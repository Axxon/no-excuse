# Code de conduite

Les échanges du projet doivent rester respectueux, inclusifs et centrés sur les faits. Le harcèlement, les attaques personnelles, la discrimination et la publication de données privées ne sont pas acceptés.

Les mainteneurs peuvent modérer ou refuser toute contribution qui met en danger des candidats, introduit un critère discriminatoire ou compromet la confidentialité des données de recrutement.
